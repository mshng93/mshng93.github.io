FREELANCE DESIGN

Role: UX Design
Tools: Adobe Creative Suite, InVision, Shopify

Halona is a travel agency website which features exclusive vacation deals for families. The site includes a wizard to help busy parents discover vacation packages that can help them save time on trip planning.




MY ROLE

This project was originally an individual student project. The final coding was performed in pairs where my partner and I decided to move forward with her original concept. I performed the majority of the coding for the ‘Insight', ‘About Us’, and 'Reviews' tabs as well as sharing responsibility for building the style sheets.




REFLECTION

The digital medium is based off of print, but the two work in very different fashions. In a world where so many different screen sizes exists, it was a challenge to create a responsive website as several new considerations had to be accounted for when the canvas changes.

Rather than minimizing or enlarging the same elements on the page for different screen resolutions, redesigning certain elements could be more effective for a better experience.
